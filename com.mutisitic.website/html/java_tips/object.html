一、类与实例

1、对象（Object）

  • 万物皆对象：一切事物皆为对象，对象就是可以看到、感觉到、听到、尝到、闻到等的东西。
  • 对象是一个自包含的实体，用一组可识别的特性和行为来标识。
  • 面向对象程序设计（OOP，Object-Oriented Programming）其实就是就是针对对象来进行编程的意思

2、类（Class）

  • 类就是具有相同的属性和功能的对象的抽象的集合
  • 类是对象的抽象，而对象是类的具体实例
  • 类（Class）是面向对象程序设计（OOP，Object-Oriented Programming）实现信息封装的基础。类是一种用户定义类型，也称类类型。每个类包含数据说明和一组操作数据或传递消息的函数。类的实例称为对象
  • Class是表示类的关键字，类名首字母要大写。采用驼峰命名法

3、实例（Instance）

  • 类是创建实例的模板，对象代表了类的一个特定的实例，各个实例拥有的数据都互相独立，互不影响
  • 对象就是类的实例，所有的对象都是实例，但并不是所有的实例都是对象
  • 实例化就是创建对象的过程，使用 new 关键字来创建。eg：Object obj = new Object();
